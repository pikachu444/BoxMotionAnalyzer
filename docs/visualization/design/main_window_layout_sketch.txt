### Component Breakdown:

*   **main_window.py**:
    *   Manages the main window, menu bar, and status bar.
    *   Assembles the visualization widgets into the main layout.

*   **vista_widget.py**:
    *   The left-hand side panel containing the `pyvistaqt.QtInteractor`.
    *   Responsible for all 3D rendering logic.

*   **control_panel.py**:
    *   Holds the right-side interactive controls.
    *   **Display Options**: `QCheckBox` widgets to toggle box, markers, and labels.
    *   **Scene Inspector**: A grouped `QTreeWidget` that separates `Center of Mass`, `Corners`, and `Markers`.
    *   **Metric Selector**: A `QComboBox` whose entries change based on the selected entity type.
    *   **Metric Help Label**: A short inline label that explains the supported frames and metrics for the selected entity type.
    *   **Frame Range Filter**: `Use Frame Range`, `Start`, `End` controls for plot filtering.

*   **plot_widget.py**:
    *   Displays 2D graphs using `matplotlib`.
    *   Receives data based on the selection in the `Scene Inspector`.

*   **info_log_widget.py**:
    *   Displays the `Frame Inspector` table for the current frame.
    *   The visible rows vary by selected entity type.
    *   Marker selections expose position rows only, while CoM selections also expose global and box-local velocity/acceleration rows.

*   **data_handler.py** (Non-visual):
    *   Reads the exported multi-header result CSV.
    *   Builds the visualization long-format DataFrame and classifies entities as `CoM`, `corner`, or `marker`.
