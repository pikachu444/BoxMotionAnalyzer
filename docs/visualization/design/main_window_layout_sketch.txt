### Component Breakdown:

*   **main_window.py**:
    *   Manages the main window, menu bar, and status bar.
    *   Assembles all other widgets into the main layout (e.g., `QHBoxLayout`).

*   **vista_widget.py**:
    *   The left-hand side panel containing the `pyvistaqt.QtInteractor`.
    *   Responsible for all 3D rendering logic.

*   **control_panel.py**:
    *   A composite widget that will eventually hold all interactive controls.
    *   **Display Options**: A group of `QCheckBox` widgets to toggle the visibility of different actors (box, markers, labels).
    *   **Object Inspector**: A `QListWidget` populated with all unique `object_id`s from the loaded data. Used to select an object for analysis.
    *   **Information Log**: A `QTextEdit` (read-only) to display detailed numerical data of the selected object or frame.
    *   **Animation Controls**: `QPushButton`s for play/pause and a `QSlider` for timeline scrubbing.

*   **plot_widget.py**:
    *   The panel for displaying 2D graphs using `matplotlib`.
    *   It will receive data based on the selection in the `Object Inspector`.

*   **data_handler.py** (Non-visual):
    *   The central hub for all data operations. Triggered by user actions in the GUI.
