# Box Motion Analyzer

모션 캡처 데이터에서 박스의 움직임을 분석하기 위한 GUI 애플리케이션입니다.

## 실행 방법

애플리케이션을 실행하려면 프로젝트 루트 디렉토리에서 `run.py` 스크립트를 실행하세요:

## 빌드 방법 (배포용)

이 프로젝트는 `PyInstaller`를 사용하여 독립 실행형 실행 파일로 빌드할 수 있습니다. `build.bat` 파일을 사용하면 간편하게 빌드할 수 있습니다.

### 1. `build.bat` 실행
`build.bat` 파일을 더블 클릭하거나 터미널에서 실행합니다.

### 2. 빌드 모드 선택
스크립트 실행 시 두 가지 빌드 모드 중 하나를 선택할 수 있습니다:

*   **1. 폴더 방식 (onedir) - 권장**:
    *   실행 파일과 의존성 파일들이 하나의 폴더(`dist/BoxMotionAnalyzer/`)에 생성됩니다.
    *   빌드 속도가 빠르고, 프로그램 실행 속도도 빠릅니다.
    *   개발 및 내부 테스트 용도로 적합합니다.

*   **2. 단일 파일 방식 (onefile)**:
    *   모든 파일이 하나의 `.exe` 파일(`dist/BoxMotionAnalyzer.exe`)로 압축됩니다.
    *   배포가 간편하지만, 실행 시 임시 폴더에 압축을 푸는 과정이 있어 시작 속도가 다소 느릴 수 있습니다.
    *   최종 사용자 배포 용도로 적합합니다.

## 주요 기능

*   **데이터 시각화 (Data Visualization):** 모션 캡처 데이터를 로드하고 시각화합니다.
*   **분석 파이프라인 (Analysis Pipeline):** 데이터 처리 파이프라인을 실행하여 박스 모션을 분석합니다.
*   **시나리오 내보내기 (Scenario Export):** 분석 결과를 CSV 파일로 내보냅니다.
    *   **자동 오프셋 계산:** 분석 시나리오를 위해 가장 낮은 3개의 코너를 자동으로 선택합니다.
    *   **수동 오프셋 선택:** 3개의 코너를 수동으로 선택할 수 있습니다.
    *   **수동 높이 입력:** 선택된 코너의 높이 값을 수동으로 지정하여 계산된 값을 덮어쓸 수 있습니다.
