# Analysis GUI Mockups Notes

- 현재 Step 1 / Step 2 UI 흐름을 설명하는 이미지와 생성 스크립트를 관리한다.

- 구현 코드가 아니라 설계 보조 자료이므로 `docs/analysis/design/` 아래에 둔다.
- 이 스크립트와 이미지들은 임시 참고용 자료이며, 현재 구현의 기준은 `src/analysis/` 와 현재 상태 문서다.
- mockup이 현재 UI와 달라지면 이미지와 생성 스크립트를 함께 갱신한다.
