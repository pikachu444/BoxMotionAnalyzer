## 1. 프로젝트 목표

실험 데이터(CSV)로부터 3D 객체(박스, 마커 등)의 모션을 분석하고, 이를 상호작용 가능한 GUI 환경에서 시각화하는 애플리케이션을 개발한다.

## 2. 기술 스택

- **GUI Framework**: PySide6
- **3D Visualization**: PyVista, pyvistaqt
- **Data Handling**: pandas, NumPy
- **2D Plotting**: Matplotlib

## 3. 아키텍처 및 소스 코드 구조

- **`main.py`**: 애플리케이션의 시작점.
- **`main_window.py`**: 메인 GUI 창. 모든 UI 위젯(`ControlPanel`, `PlotWidget` 등)을 조립하고, 시그널-슬롯 연결을 총괄하는 중앙 컨트롤러.
- **`vista_widget.py`**: PyVista 렌더링 위젯. 3D 객체 생성 및 업데이트 로직을 담당.
- **`control_panel.py`**: 사용자의 입력을 받는 모든 제어 위젯(객체 선택, 옵션 체크박스 등)을 담는 컨테이너.
- **`plot_widget.py`**: Matplotlib 그래프를 표시하는 위젯.
- **`info_log_widget.py`**: 선택된 객체의 상세 정보를 테이블 형태로 표시하는 위젯.
- **`data_handler.py`**: 데이터 처리 클래스 (CSV 읽기/변환, 데이터 조회).
- **`config.py`**: 프로젝트 전역 설정 및 상수.
- **`gui_layout_sketch.txt`**: GUI 레이아웃 설계도.
