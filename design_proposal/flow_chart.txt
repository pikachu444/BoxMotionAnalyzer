=========================================
|   GUI 분석기 프로그램 흐름도          |
=========================================

(시작)
   |
   V
[ 1. 사용자가 프로그램을 실행 ]
   |
   +-----> [ PySide6 기반의 메인 GUI 창이 나타남. 대부분의 위젯은 비활성화 상태 ]
   |
   V
[ 2. 사용자가 "CSV 파일 불러오기..." 버튼 클릭 ]
   |
   +-----> [ 시스템 파일 열기 대화상자 실행 ]
   |
   V
[ 3. 사용자가 CSV 파일을 선택하고 확인 ]
   |
   +-----> [ 프로그램이 전체 CSV 파일을 메모리로 읽어들임 (RawData DataFrame 생성) ]
   |       [ 필수 컬럼('Time', 'Position X/Y/Z' 등) 유효성 검사 ]
   |
   +-----> [ 만약 유효성 검사 실패 시: 사용자에게 에러 메시지 표시 후 2번 단계로 복귀 ]
   |
   V
[ 4. GUI에 데이터 정보 채우기 ]
   |
   +-----> [ **플롯 옵션:** "데이터" 드롭다운 메뉴에 "강체 중심" 및 CSV에서 찾은 모든 마커 이름들을 채움 ]
   |       [ **데이터 시각화:** 기본값(예: 강체 중심의 Y-위치 vs. 시간)으로 전체 데이터에 대한 그래프를 그림 ]
   |       [ **박스 규격, 분석 실행 버튼 등** 관련 위젯들을 활성화 ]
   |
   V
[ 5. 사용자 GUI 인터랙션 (반복 루프) ]
   |
   +-----> [ A. 사용자가 "데이터" 또는 "축" 드롭다운 메뉴 변경 ]
   |           |
   |           +-----> [ **데이터 시각화:** 새로운 선택에 따라 그래프를 다시 그림 ]
   |
   +-----> [ B. 사용자가 그래프 위에 마우스 오버 ]
   |           |
   |           +-----> [ 마우스 커서 위치의 (시간, 값) 정보를 보여주는 툴팁(Tooltip) 표시 ]
   |
   +-----> [ C. 사용자가 그래프 위에서 특정 구간을 클릭 또는 드래그 (비파괴적 슬라이싱) ]
   |           |
   |           +-----> [ **(시각적 변경)** 선택된 구간이 그래프 위에 다른 색상이나 음영으로 겹쳐서 표시됨 (오버레이) ]
   |           |       [ **(값 변경)** "분석 구간"의 "시작"과 "종료" 입력 필드 값이 선택된 시간으로 자동 업데이트됨 ]
   |           |       [ (주의) 이 단계에서 파일이 생성되거나 원본 데이터가 변경되지 않음. ]
   |
   +-----> [ D. 사용자가 "박스 규격" 또는 "분석 구간" 값을 수동으로 입력 ]
   |           |
   |           +-----> [ 내부 설정값 (`AnalysisConfig` 객체)이 업데이트됨 ]
   |
   +-----> [ E. 사용자가 "분석 실행" 버튼 클릭 ]
   |           |
   |           +-----> (이 루프를 빠져나와 6번 단계로 진행)
   |
   V
[ 6. "분석 실행" 버튼 클릭됨 ]
   |
   +-----> [ **GUI 동결:** 처리 중 변경을 막기 위해 모든 입력 컨트롤을 임시 비활성화 ]
   |       [ **로그 출력:** "[정보] 분석을 시작합니다..." 메시지 표시 ]
   |       [ **진행률 표시줄:** 움직이기 시작 ]
   |
   V
[ 7. 파이프라인 실행 (`PipelineController`가 `Analysis Modules`를 순차 호출) ]
   |
   +-----> [ **7a. Slicer 실행:** GUI의 "분석 구간" 설정값으로 DataFrame 슬라이싱 ]
   |       |
   |       V
   |       [ **7b. Parser 실행:** 원본 데이터를 분석용 포맷으로 변환 ]
   |       |
   |       V
   |       [ **7c. MarkerSmoother 실행:** 마커 데이터에 스무딩 필터 적용 ]
   |       |
   |       V
   |       [ **7d. PoseOptimizer 실행:** 스무딩된 마커로부터 강체 자세 계산 ]
   |       |
   |       V
   |       [ **7e. VelocityCalculator 실행:** 자세 데이터로부터 속도/가속도 계산 ]
   |       |
   |       V
   |       [ **7f. FrameAnalyzer 실행:** 계산된 운동학 데이터를 특정 좌표계 기준으로 변환/분석 ]
   |       |
   |       V
   |       [ **최종 DataFrame 생성:** 모든 분석 결과가 누적된 최종 DataFrame이 반환됨 ]
   |       |  - [로그] "[정보] 분석 완료."
   |
   V
[ 8. 분석 완료 ]
   |
   +-----> [ **GUI 활성화:** 입력 컨트롤들을 다시 사용 가능하게 변경 ]
   |       [ **"결과 CSV로 내보내기" 버튼**이 활성화됨 ]
   |       [ **진행률 표시줄:** 100%에 도달 ]
   |
   V
[ 9. 사용자가 "결과 CSV로 내보내기" 버튼 클릭 ]
   |
   +-----> [ 시스템 "다른 이름으로 저장" 대화상자 실행 ]
   |
   V
[ 10. 사용자가 파일 이름과 위치를 선택 ]
    |
    +-----> [ 7d 단계에서 취합된 최종 분석 결과가 지정된 경로에 CSV 파일로 저장됨 ]
    |       [ **로그 출력:** "[정보] 결과가 {filepath}에 성공적으로 저장되었습니다." ]
    |
    V
[ (다음 분석을 위해 5번 단계로 복귀) ]
    |
    V
(종료)
